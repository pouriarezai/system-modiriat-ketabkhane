Project Overview:

This project is a library management system that helps manage books, members, and book loans. It includes features to add, update, and delete books, manage member information, and track book loans.

Files Included:

 1. library.py: Manages the library operations, including adding, updating, and deleting books.
 2. members.py: Handles member information, including adding and updating member details.
 3. database.db: SQLite database file used to store book and member records.
 4. main.py: Provides a command-line interface to interact with the library management system.

Usage:

 1. Run main.py to start the program.
 2. Follow the on-screen prompts to manage books, members, and book loans.

توضیحات پروژه:

این پروژه یک سیستم مدیریت کتابخانه است که به مدیریت کتاب‌ها، اعضا و امانت‌های کتاب کمک می‌کند. این سیستم شامل ویژگی‌هایی برای اضافه، به‌روزرسانی و حذف کتاب‌ها، مدیریت اطلاعات اعضا و پیگیری امانت‌های کتاب است.

فایل‌های موجود:

 1. library.py: مدیریت عملیات کتابخانه، از جمله اضافه، به‌روزرسانی و حذف کتاب‌ها.
 2. members.py: مدیریت اطلاعات اعضا، از جمله اضافه و به‌روزرسانی جزئیات اعضا.
 3. database.db: فایل دیتابیس SQLite که برای ذخیره سوابق کتاب‌ها و اعضا استفاده می‌شود.
 4. main.py: ارائه رابط کاربری خط فرمان برای تعامل با سیستم مدیریت کتابخانه.

نحوه استفاده:

 1. فایل main.py را اجرا کنید تا برنامه شروع شود.
 2. با پیروی از دستورالعمل‌های روی صفحه، کتاب‌ها و اعضا را مدیریت کرده و امانت‌های کتاب را پیگیری کنید.
